  3. Variational Formulation
    Multiply the PDE by a test function and integrate over an element

    The test function v is chosen from the same function space as the trial functions .

  4. Why use the test function ( v )?
    (시험함수 v는 기저 함수들 중 하나로 선택될 수 있음.)
    The equation:
    [
    \int*{\Omega_e} -\nabla^2 u v , dx = \int*{\Omega_e} f v , dx
    ]
    involves multiplying by a test function ( v ). The reason for this is to construct ( N+1 ) equations to solve ( N+1 ) unknowns in the problem (as seen in Equation 2.3). By using ( N+1 ) different test functions, you obtain ( N+1 ) linearly independent equations.

  5. Transforming to weak form
    The requirement that ( u ) must be twice differentiable in the strong form is restrictive. To relax this condition, the equation is rewritten in its weak form, where the test and trial functions only need to be once differentiable.

  6. Using Gauss-Green’s lemma
    Applying Gauss-Green’s theorem, the equation can be reformulated as:
    [
    \int*{\Omega_e} -\nabla^2 u v , dx = \int*{\Omegae} \nabla u \cdot \nabla v , dx - \int{\partial \Omegae} \frac{\partial u}{\partial n} v , dS
    ]
    The boundary term ( \int    {\partial \Omegae} \frac{\partial u}{\partial n} v , dS ) often vanishes due to boundary conditions, simplifying the equation to:
    [
    \int    {\Omega_e} \nabla u \cdot \nabla v , dx
    ]

  7. Conclusion
    This process is fundamental to the finite element method. Transforming to the weak form makes the problem easier to solve numerically by reducing the differentiability requirements on the solution and test functions.
